# clickScript
用于巨量百应直播中控台 自动点击讲解脚本
每隔15秒点击一次讲解按钮

## 使用方法
### 一、油猴脚本（推荐）

1. 安装油猴插件 [油猴插件——安装与使用教程](https://zhuanlan.zhihu.com/p/128453110)
2. 将clickScriptPlugin.js文件中的代码复制到油猴中
![plugin1.png](score%2Fplugin1.png)
3. Ctrl+s保存脚本
4. 回到巨量百应界面，刷新页面
